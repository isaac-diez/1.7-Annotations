📄**Description**

Task 1.7 is about practicing and learning how the Annotations in Java work.


### English
- Exercise 1: Create a customized annotation that should allow a Java object to be inserted into a JSON file. The annotation must receive the directory where the resulting file will be placed.

### Català
- Exercici 1 :Crea una anotació personalitzada que ha de permetre serialitzar un objecte Java en un fitxer JSON. L’anotació ha de rebre el directori on es col·locarà el fitxer resultant.
